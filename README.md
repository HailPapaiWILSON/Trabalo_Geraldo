# Projeto de Pesquisa

## Tema Principal

**IA no Desenvolvimento de Software**

---

## Título

**O Impacto das Ferramentas de IA Generativa na Produtividade do Desenvolvimento de Software: Uma Revisão Bibliográfica**

---

## Pergunta de Pesquisa

**Como as ferramentas de IA generativa impactam a produtividade no desenvolvimento de software e quais fatores influenciam a magnitude desses ganhos?**

---

## Resumo Técnico

O uso de ferramentas de inteligência artificial generativa no desenvolvimento de software tem crescido rapidamente nos últimos anos, transformando a forma como programadores escrevem, revisam e compreendem código. Soluções como GitHub Copilot e ChatGPT passaram a integrar o cotidiano de profissionais e organizações, prometendo aumentar a produtividade e reduzir o tempo necessário para a realização de diversas tarefas de programação. Entretanto, os resultados apresentados na literatura científica ainda não são totalmente consensuais, indicando que os impactos dessas ferramentas podem variar de acordo com fatores como experiência do desenvolvedor, tipo de atividade realizada e contexto de aplicação.

Diante desse cenário, este trabalho investiga o impacto das ferramentas de IA generativa na produtividade do desenvolvimento de software a partir da análise de estudos recentes publicados na área. A revisão da literatura busca identificar evidências sobre ganhos de desempenho, mudanças no fluxo de trabalho dos desenvolvedores e limitações associadas ao uso dessas tecnologias. Além disso, são discutidos conceitos fundamentais relacionados à inteligência artificial generativa, modelos de linguagem de grande escala e métricas de produtividade em engenharia de software, fornecendo uma base teórica para a compreensão dos resultados encontrados nos estudos analisados.
