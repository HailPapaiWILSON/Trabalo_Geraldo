# Projeto de Pesquisa

## Tema Principal

**IA no Desenvolvimento de Software**

---

## Título

**O Impacto das Ferramentas de IA Generativa na Produtividade do Desenvolvimento de Software: Uma Revisão Bibliográfica**

---

## Pergunta de Pesquisa

**Como as ferramentas de IA generativa impactam a produtividade no desenvolvimento de software e quais fatores influenciam a magnitude desses ganhos?**

---

## Resumo Técnico

O uso de ferramentas de inteligência artificial generativa no desenvolvimento de software tem crescido rapidamente nos últimos anos, transformando a forma como programadores escrevem, revisam e compreendem código. Soluções como GitHub Copilot e ChatGPT passaram a integrar o cotidiano de profissionais e organizações, prometendo aumentar a produtividade e reduzir o tempo necessário para a realização de diversas tarefas de programação. Entretanto, os resultados apresentados na literatura científica ainda não são totalmente consensuais, indicando que os impactos dessas ferramentas podem variar de acordo com fatores como a experiência do desenvolvedor, o tipo de atividade realizada e o contexto de aplicação.

Compreender esse cenário exige a análise de conceitos fundamentais relacionados aos modelos de linguagem de grande escala (LLMs) especializados em código e às métricas contemporâneas de produtividade em engenharia de software. A investigação técnica do tema orienta-se pela necessidade de identificar evidências empíricas sobre ganhos reais de desempenho e mudanças estruturais no fluxo de trabalho. Além disso, o mapeamento dos fatores que influenciam a magnitude desses ganhos — bem como as limitações associadas a alucinações de código, segurança e dependências obsoletas — estabelece a base teórica necessária para avaliar a transição do desenvolvedor tradicional para um papel de revisor crítico de código gerado por IA.
